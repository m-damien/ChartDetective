How To Use: 3 Steps
1

Select the chart to extract through a rectangular selection.

Only vector charts can be selected. If the selection fails, you need a tool that works with rasterized images such as WebPlotDigitizer
2

Select the Y axis and then drag it into the corresponding table. Repeat for the X axis.

3

Select marks that you want to extract (bars, lines, dots, box plots) and drop them in the corresponding zone. You may use colour and shape filters to ease selections
